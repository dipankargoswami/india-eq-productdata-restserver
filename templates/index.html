BhavCopy Data

The API provides Bhavcopy data for any specific stock in NSE & BSE.

End Point: /api/v1/bhavcopy/

Parameter Description Sample values Mandatory
exchange Exchange for which data is needed Following are the valid values
  • NSE
  • BSE
    		•  Y
    stock Stock for which data is needed
  • NSE-
    •   Symbol. e.g. INFY
  • BSE-
    •   Numeric Scrip code. e.g. 500002     		Y


    Example: /api/v1/bhavcopy/?exchange=NSE&stock=INFY
